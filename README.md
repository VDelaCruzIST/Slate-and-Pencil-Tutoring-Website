# Slate & Pencil Tutoring Website

## Overview
This repository contains a tutoring services website developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The website promotes an online tutoring service and demonstrates modern web development concepts including CSS Grid layouts, navigation systems, content organization, and testimonial sections.

## Features
* Educational services website design
* CSS Grid page layout
* Vertical and horizontal navigation menus
* Subject category navigation
* Student testimonial sections
* Custom graphics and image integration
* Semantic HTML5 structure

## Technologies Used
* HTML5
* CSS3
* CSS Grid Layout

## Project Files
* `sp_home.html` – Main website page
* `sp_base.css` – Base styles
* `sp_styles.css` – Typography and visual styling
* `sp_layout.css` – CSS Grid layout and page structure
* Student and topic image assets
* Website logo

## Skills Demonstrated
* Front-end web development
* CSS Grid implementation
* Website navigation design
* Layout and content organization
* Responsive page structuring
* Asset management
* Semantic HTML development

This project was completed as part of web development coursework and demonstrates foundational front-end development skills using HTML and CSS.
